  • Write your template in a string in a Javascript file
  • Use JSX in a Javascript file

Directives

v-bind

<!-- full syntax -->
<a v-bind:href="url"></a>
<!-- shorthand -->
<a :href="url"></a>
v-on

<!-- full syntax -->
<a v-on:click="doMethod"></a>
<!-- shorthand -->
<a @click="doMethod"></a>
Modifiers

<form v-on:submit.prevent="onSubmit"> ... </form>
